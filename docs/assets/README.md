# Logo Assets

Logo files for the Undash-cop Metrics Billing Platform.

## Logo Files

### `logo.svg`
Full horizontal logo with icon and text. Use for headers, documentation, and marketing materials.
- Dimensions: 200x60px
- Format: SVG (scalable)

### `logo-horizontal.svg`
Extended horizontal logo with gradient background. Use for wide headers and hero sections.
- Dimensions: 300x80px
- Format: SVG (scalable)

### `logo-icon.svg`
Square icon-only logo. Use for app icons, favicons, and square spaces.
- Dimensions: 64x64px
- Format: SVG (scalable)

### `logo-favicon.svg`
Small favicon version. Use for browser tabs and bookmarks.
- Dimensions: 32x32px
- Format: SVG (scalable)

## Usage

### In HTML/Markdown
```html
<img src="/assets/logo.svg" alt="Undash-cop Metrics Billing Platform" />
```

### In Documentation
```markdown
![Undash-cop Logo](/assets/logo.svg)
```

### As Favicon
```html
<link rel="icon" type="image/svg+xml" href="/assets/logo-favicon.svg" />
```

## Design Elements

- **Colors**: 
  - Primary Blue: `#1e40af` (blue-800)
  - Secondary Blue: `#3b82f6` (blue-500)
  - Light Blue: `#60a5fa` (blue-400)
  - White: `#ffffff`
  - Green: `#10b981` (green-500) - for checkmark/approval

- **Icons**:
  - Chart/Graph bars representing metrics
  - Trend line showing growth
  - Invoice/document representing billing
  - Checkmark representing completion/approval

## Brand Guidelines

- Always maintain aspect ratio when resizing
- Minimum size: 100px width for horizontal logos
- Use SVG format for best quality at all sizes
- Do not modify colors without brand approval
- Maintain clear space around logo (minimum 20% of logo height)

## Export Formats

If you need PNG versions, export from SVG at these sizes:
- Logo: 400x120px (2x), 600x180px (3x)
- Icon: 128x128px (2x), 256x256px (3x)
- Favicon: 64x64px (2x), 128x128px (3x)
